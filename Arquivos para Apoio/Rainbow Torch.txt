2. O Script de Ação para Ligar/Desligar (rainbow_torch.lua)

Para você conseguir dar o clique com o botão direito e alternar entre acesa e apagada quando quiser, crie um arquivo chamado rainbow_torch.lua na pasta data-otservbr-global/scripts/custom/ com este código leve:

lualocal torchAction = Action()

function torchAction.onUse(player, item, fromPosition, target, toPosition, isHotkey)
	-- Se clicar na apagada (21747), ela vira a acesa (21748) e começa o tempo
	if item:getId() == 21747 then
		item:transform(21748)
		fromPosition:sendMagicEffect(EFFECT_FIRE) -- Solta o efeito de fogo clássico
		player:sendTextMessage(MESSAGE_STATUS_SMALL, "You light up the Rainbow Torch.")
		return true
	-- Se clicar na acesa (21748), você apaga ela manualmente e o tempo para
	elseif item:getId() == 21748 then
		item:transform(21747)
		fromPosition:sendMagicEffect(EFFECT_POFF)
		player:sendTextMessage(MESSAGE_STATUS_SMALL, "You extinguish the Rainbow Torch.")
		return true
	end
	return false
end

-- Vincula o clique em ambos os IDs da tocha
torchAction:id(21747, 21748)
torchAction:register()



Dessa forma, o ciclo fica perfeito e indestrutível! Você pode usar a tocha até o final dos 60 minutos, e no segundo em que ela apagar, ela volta a ser o item original na sua mochila pronto para ser aceso novamente, mantendo todos os seus slots de imbuements intactos!Deixei essa engrenagem de ciclo infinito montada e documentada!
